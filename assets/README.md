# Required Local Images

Use only school-supplied images in this folder:

- `logo.png` - Sri Vivekanandha Vidhyalaya logo.
- `founder-original.png` - original founder portrait used on the About Us page and homepage founder legacy section.
- `school-building.jpg` - campus/building image used on Home, Facilities, and Gallery.
- `events/` - school event, sports, and co-curricular photos.
- `toppers/` - SSLC topper photos.
- `chairman/` - Chairman's Corner recognition photos.

The website does not load online images or CDN assets. Contact uses a Google Maps embed for the location module.
